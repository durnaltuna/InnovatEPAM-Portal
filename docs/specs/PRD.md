# PRD: InnovatEPAM Portal (Phase 1 MVP)

## Vision
A centralized platform for EPAMers to submit and manage innovation ideas.

## Target Audience
- **Submitters**: EPAM employees sharing ideas.
- **Admins**: Reviewers and decision-makers.

## MVP Features (Phase 1)
### 1. User Management
- Authentication (Login/Logout/Register).
- Role-Based Access Control (Submitter vs. Admin).
### 2. Idea Submission
- Form: Title, Description, Category.
- Attachment: Support for one file attachment per idea.
- Dashboard: List view of "My Submissions".
### 3. Evaluation Workflow
- Status Tracking: Submitted -> Under Review -> Accepted/Rejected.
- Admin Panel: View all submissions and change status with comments.
